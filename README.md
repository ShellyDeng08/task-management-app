### run at local

1. clone this code to your local:
```
git@github.com:ShellyDeng08/task-management-app.git
```

2. install dependenceies:
```
npm install
```

3. run at local:
```
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
